# Usage
<!-- usage -->
```sh-session
$ npm install -g stackmate
$ stackmate COMMAND
running command...
$ stackmate (-v|--version|version)
stackmate/0.1.0 darwin-x64 node-v12.16.2
$ stackmate --help [COMMAND]
USAGE
  $ stackmate COMMAND
...
```
